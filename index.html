Nesbox Emulator

Nesbox Emulator is a free multiplatform emulator of NES, SNES, SEGA, GAMEBOY video consoles.

Release Notes

Version 1.2 22.05.2015

  • Added setup gamepad page where you can change position of the buttons

  • Fixed some app crashes

Version 1.1 08.05.2015

  • Supported video consoles: NES, Super Nintendo, Sega Genesis/MegaDrive, GameBoy Color/Advanced
  • Supported rom types: .nes .gen .smd .smc .gb .gbc .gba
  • Save/Load game state
  • The Library of added games
  • Keyboard support (allows to changed key bindings)
  • Screen gamepad (touch)
  • Xbox 360 Controller support
  • Search games over Internet
  • Supported platforms: Windows Store, Windows Phone, Adobe Flash (open source)
